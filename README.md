# 你不懂JS（系列丛书）

这是一套深入探讨 JavaScript 语言核心机制的系列丛书。本系列的第一版已经完成。

## 本地阅读

首先保证本地已安装有 [Node.js](http://nodejs.cn/download/) 环境。

```bash
npm i -g nrm

# 切换国内的 npm 镜像
nrm use taobao
```

```bash
# 安装 docsify-cli
npm i -g docsify-cli

# 在项目根目录启动服务
docsify serve docs
```
打开浏览器，访问 <http://localhost:3000> 开始读书吧~
